# PCR Machine Technology Dashboard

A high-fidelity, professional product specification page for the **CFX96 Bio-Rad Real-Time PCR Machine**. This project was developed as a technical internship task to demonstrate modern web development skills in the healthcare/medical UI domain.

## ✨ Key Features
* **Modern Healthcare UI:** A clean, laboratory-grade interface using a professional color palette (Navy Blue #2d5da1 & Clinical Red #e52730).
* **Sticky-Visual Layout:** The product image remains fixed on the left while the technical data scrolls on the right, providing constant visual context.
* **Fully Responsive:** Optimized for all screen sizes (Desktop, Tablet, and Mobile) using CSS Media Queries.
* **Interactive Elements:** Subtle hover effects and scale transitions on feature cards for an improved user experience.

## 🛠️ Technical Stack
* **HTML5:** Semantic structure for better accessibility.
* **CSS3:** Advanced Flexbox and CSS Grid for a robust layout.
* **CSS Variables:** Used for global color and spacing management (Atomic Design principles).
* **Typography:** Integrated Google Fonts (Inter) for a premium look.

## 📁 Project Structure
* `index.html` - The main entry point containing the structure and styles.
* `images.jpg` - High-resolution product photography of the CFX96 PCR Machine.

## 📝 How to Run Locally
1. Clone this repository or download the ZIP file.
2. Ensure the `images.jpg` file is in the same folder as `index.html`.
3. Open `index.html` in any modern web browser (Chrome, Edge, or Firefox).
